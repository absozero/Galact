Galact

An Open Source chatbot, in Python.

Docs!

Open source

Galact is completely free and open source. This makes the development process much more transparent, and it is easier to contribute to and improve the project faster. Its code can be accessed at the github!

Easy to Use

Galact is very easy to install and can be installed with pip, works like most other cli apps. It has a built in argument parser and works extensively with the use of argparse.

CLI-Infused

Galact works very well on a command line and can be easily used by anybody, due to simplistic installment and easy and fast usage. It is even being planned to port to a gui!
